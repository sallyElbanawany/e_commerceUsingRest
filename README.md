# E-Commerce App 
### An E-Commerce build Using JAX-RS REST API and JAX-WS SOAP Web Service

## Technologies Used
* `JAX-WS`
* `JPA` 
* `MySql`
* `Maven`
* `Tomcat`
-------------------------

## Run with Maven
**Maven**
* Change the configuration in `pom.xml` (tomcat username , password and port number). 
* Deploy the application using maven command:
 ```
mvn clean package tomcat7:redeploy
```

**My SQL**
* Create a database schema and change (username , password and database name) in the persistence.xml
